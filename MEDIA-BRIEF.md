# COSTE — Website Photography Brief

**For:** Media Manager
**From:** Web team
**Re:** Images required for the COSTE website (coste_sidi_bou)
**Goal:** Replace placeholder images with high-quality, on-brand photography sized for crisp display on all screens (including high-resolution "Retina" displays).

---

## 1. General standards (please read first)

| Item | Standard |
|---|---|
| **File format** | JPG for photos (quality 80–85%). WebP also welcome if you can export it. PNG only for graphics/logos. |
| **Color** | sRGB color profile. Natural, warm, Mediterranean tone — avoid heavy filters. |
| **Resolution** | Always send the **largest original** you have. Target pixel sizes below are the *minimum* for sharpness; bigger is fine, we compress. |
| **File size** | Aim under ~500 KB for full-width images, ~250 KB for cards (we can optimize — don't over-compress yourself; send quality originals). |
| **Orientation** | Respect the **aspect ratio** listed for each slot. If the ratio is wrong the image gets cropped automatically and may cut off heads/plates. |
| **Composition** | Keep the main subject **centered or slightly high**. The site adds dark gradients at the **bottom** of many cards and text on the **bottom-left of the hero** — leave that area calmer. |
| **Naming** | Use clear names, e.g. `hero-terrace.jpg`, `exp-brunch.jpg`, `event-2026-07-12-jazz.jpg`. |
| **Rights** | Confirm we own / are licensed for each image (especially the "Nearby" tourist shots). |

> **Aspect ratio cheat:** 16:9 = wide landscape · 4:3 = standard landscape · 3:4 & 4:5 = portrait · 1:1 = square · 21:9 = ultra-wide banner.
